# Jewellery Inventory WordPress Plugin

This plugin is extracted from a real Jewellery POS system and provides
basic inventory management inside WordPress admin.

## Features
- Add / view jewellery items
- Inventory stock management
- Secure PHP and MySQL logic
- Built using WordPress hooks and $wpdb

## Technologies Used
- PHP
- WordPress
- MySQL

## Purpose
Created to demonstrate WordPress plugin development and open-source contribution readiness.
